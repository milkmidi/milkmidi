### 🧠 Harness Engineering

A framework for governing AI-assisted development using Guides (feedforward control) + Sensors (feedback control):

```
Guide  → CLAUDE.md / Team prompting standards
Sensor 1 → ESLint / TypeScript (static analysis)
Sensor 2 → CI/CD + Claude Code (AI code review / security scan)
Sensor 2.5 → e2e testing in a separate repo
Sensor 3 → CODEOWNERS (human-in-the-loop architecture review)
```
